HTML Terms

HTML
HyperText Markup Language it defines the meaning and structure of web content.It is a file with .html extension.
HTML element
The component of a html document that tels web browser how to structure and interpret a Html document,include all content inside the tag.
HTML tag
The HTML tag is a keywords inside HTML document and includes opening and closing tag.
angle bracket
Angle Brackets are used to enclose a code,or a HTML tag.
HTML attribute
HTML attribute provide additional information about specific element within a tag.
HTML comment
HTML comment is a coment that is not showed on the web page,but in source code, it is used to explain the code.
HTML object element
HTML object element tag represents a container for an external resource.
HTML special character
HTML special character are used to represent an HTML characters that cannot typed directly in HTML document such as &,>.
